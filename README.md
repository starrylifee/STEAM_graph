# 막대그래프 STEAM 게임

학생 설계 PDF `source/20260611075405.pdf`를 바탕으로 만든 Vanilla HTML/CSS/JS 미니게임 모음이다.

## 구성

* `index.html` - 대시보드와 게임 화면
* `style.css` - 반응형 UI와 게임 장면 스타일
* `app.js` - 다섯 게임 로직, 원본 설계 뷰어, 효과음
* `extracted_games.md` - PDF 판독 내용 요약
* `source_images/` - PDF 원본 페이지 이미지
* `source_images_review_alt/` - 보기 쉽게 회전한 설계 이미지

## 실행

`index.html`을 브라우저에서 열면 바로 실행된다. 별도 빌드나 설치가 필요 없다.

각 게임 화면의 `학생 설계 보기`에서 해당 게임의 설계 페이지를 볼 수 있다. `읽기용`은 판독을 위해 회전한 이미지이고, `원본`은 PDF에서 그대로 뽑은 원본 스캔 이미지다. `PDF 열기`로 전체 원본 PDF도 확인할 수 있다.

로켓 게임은 화면 버튼과 키보드 방향키를 모두 지원한다.
